# Project_evo
Симуляция эволюции ботов на двумерной сетке. Боты живут, двигаются, собирают ресурсы, охотятся, размножаются и мутируют. Поведение каждого бота задаётся геномом — списком команд.

<img width="1917" height="1032" alt="image" src="https://github.com/user-attachments/assets/e32a1214-1dfd-48e4-ac5c-1285a35d889b" />

## Возможности
генетические алгоритмы (наследование, мутации);

несколько источников энергии: солнце, минералы, охота, органика;

команды: движение, фотосинтез, деление, охота и др.;

одноклеточные и многоклеточные организмы;

визуализация - графиков популяции, энергии, возрастов, полная отрисовка мира в режиме распределения энергии;

экспорт логов симуляции в текстовые файлы.

## Технологии
Python 3

PyQt6 — графический интерфейс

pyqtgraph — динамические графики

стандартные библиотеки (random, dataclasses)

## Запуск
Установить зависимости:

bash
pip install -r requirements.txt

Запустить файл run.bat
